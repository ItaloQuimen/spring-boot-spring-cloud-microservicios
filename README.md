# spring-boot-spring-cloud-microservicios

Proyecto de aprendizaje orientado a la construcción de microservicios con Spring Boot y Spring Cloud.

El objetivo del repositorio es agrupar distintos microservicios dentro de un monorepo para practicar conceptos como comunicación entre servicios, configuración independiente y organización de proyectos basados en Spring.

## Estado del proyecto

Proyecto en etapa inicial de desarrollo y aprendizaje. La estructura base del monorepo ya está creada y los microservicios se irán ajustando a medida que avance la práctica.

## Microservicios

Actualmente el monorepo contiene:

- `msproducts`: microservicio de productos.
- `msitems`: microservicio de items.

## Tecnologías

- Java 17
- Spring Boot
- Spring Cloud
- Maven
- MySQL

## Estructura

```text
spring-boot-spring-cloud-microservicios/
├── msproducts/
└── msitems/
```

## Nota

Este repositorio tiene fines de estudio. Puede cambiar con el tiempo mientras se incorporan nuevos conceptos, configuraciones y microservicios.
